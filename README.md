# Jeu casse-briques

Jeu vidéo de type casse-briques développé avec Processing (Java).

## Objectif du projet
Ce projet m’a permis de travailler la logique de jeu, la gestion des collisions, le score et l’affichage graphique.

## Technologies utilisées
- Java
- Processing

## Fonctionnalités
- déplacement de la raquette
- balle dynamique
- collisions avec les briques
- système de score
- fin de partie

## Lancer le projet
1. Installer Processing
2. Ouvrir le fichier principal du projet
3. Cliquer sur "Run"

Ce projet a été réalisé dans le cadre de mon apprentissage de la programmation.
